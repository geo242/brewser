*A quick note on accuracy

Browser capability detection is pretty simple, but as you probably know device detection is a bit of a pain. brewser.js sniffs the User Agent string to detect certain device types and operating systems. That is to give you as much useful information as possible to distinguish between key device and OS types. As we know this can be tricky and inaccurate in most cases, please #handlewithcare.
